# FNP Sales Dashboard Analysis

## Project Overview

This project presents an interactive sales analytics dashboard developed in Microsoft Excel to analyze business performance for FNP (Ferns N Petals).

The dashboard provides insights into revenue trends, customer spending patterns, product performance, delivery efficiency, occasion-based sales, and city-wise order distribution.

---

## Tools & Technologies

* Microsoft Excel
* Power Query
* Power Pivot
* DAX (Data Analysis Expressions)
* Pivot Tables
* Pivot Charts
* Slicers & Timelines

## Business Insights

### Revenue Trends

Revenue exhibits strong seasonality, with significant peaks during gifting and festive periods.

### Top Categories

* Colors
* Soft Toys
* Sweets

### Top Products

* Magnum Set
* Quia Gift
* Dolores Gift
* Harum Pack
* Deserunt Box

### Top Cities

* Imphal
* Dhanbad
* Kavali
* Haridwar
* Bidhanagar

### Occasion Performance

Anniversary, Raksha Bandhan, Holi, and Birthday occasions generate the highest sales.
